# WhatsApp Groups — Booklesss Marketing Tracker

Track every group, every post, and every conversion here.
Update after each drop. This is the live record.

---

## How to use this file

1. Add each group before you post to it
2. After posting: log the date, which lead magnet version, and the bit.ly link used
3. After 48h: log click count from bit.ly
4. Log any Slack joins you can attribute to that group

---

## Tracking link format

Each group gets a unique bit.ly link pointing to the Slack invite.
Format: `https://bit.ly/booklesss-[group-slug]`

Create each link at bit.ly before posting. Name them clearly so the dashboard is readable.
