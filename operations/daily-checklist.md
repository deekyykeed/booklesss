# Booklesss — Daily Operations Checklist

Keep this open. Work top-to-bottom. Every item you skip today is a conversion you miss tomorrow.

---

## Every Day (15–30 min)

### Morning — Content
- [ ] Check `marketing/groups.md` — any groups you haven't posted to yet?
- [ ] If new group → create bit.ly link → add to `groups.md` → drop the lead magnet
- [ ] Check bit.ly dashboard — log click counts for any links >48h old
- [ ] Reply to any WhatsApp DMs from the lead magnet within 2 hours

### Morning — Slack
- [ ] Check #bbf4302-notes, #bbf4302-discussion, #bbf4302-quiz for unanswered questions
- [ ] Reply to every question in discussion channels (even "good question — see Module X")
- [ ] React ✅ to any module posts that new students just read (acknowledges them)

### Evening — Growth
- [ ] Did anyone DM you on WhatsApp today? → Log in `operations/leads.md`
- [ ] Did any lead magnet reader join Slack? → Update `marketing/groups.md` → Joins column
- [ ] Did any free trial member upgrade? → Update student count, note in `operations/revenue-log.md`

---

## Every Wednesday

- [ ] Drop the weekly quiz in `#bbf4302-quiz` (use `/generate-quiz` if not already written)
- [ ] Tag `@bbf4302-students` with the quiz post
- [ ] Post reminder in any WhatsApp groups where you've been active this week

---

## Every Sunday

- [ ] Tally all points from the week (quiz scores + ✅ reactions + discussion contributions)
- [ ] Update the Google Sheet — master points tab
- [ ] Format and post leaderboard to `#leaderboard` at 7pm (use `/weekly-leaderboard`)
- [ ] Post top 3 in any active WhatsApp groups as social proof ("Week N results — [name] took the top spot 🏆")

---

## Every Monday (Weekly Review — 30 min)

### Content pipeline
- [ ] Which steps are written and ready? Which are missing?
- [ ] Write the next step if needed → use `/write-module-notes`
- [ ] Any step ready for PDF? → use `/booklesss-pdf`
- [ ] Any step ready to post to Slack? → use `/post-module-to-slack`

### Marketing
- [ ] Review `marketing/groups.md` — click rates, conversion rates per group
- [ ] Identify 3 new groups to post to this week
- [ ] Is the lead magnet still accurate? (pricing, deadline, Slack link) → update if needed
- [ ] WhatsApp post template still converting? If CTR <5% → test a new hook

### Numbers
- [ ] How many active students on Slack right now?
- [ ] How many free trial → paid conversions this week?
- [ ] Revenue this month vs last month
- [ ] Slack cost this month (Business+ billing page) — still in margin?
