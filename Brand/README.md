# Brand/ — Raw Asset Drop Zone

Drop zone for raw brand assets (logos, hero images, exports) before they're organized into the project. Mirrors the Partnr Bucket pattern.

**Nothing in here is consumed by build scripts.** Build-consumed brand assets live in `_dev/brand/` (logo, mark, grain.png); fonts in `_dev/fonts/`. If a script needs an asset from here, copy it into `_dev/brand/` first.
