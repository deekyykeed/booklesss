# GoogleSans Font Family

This folder contains the GoogleSans font family used for course titles in the app.

## Available Weights

- **Regular** (`GoogleSans-Regular.ttf`) - Default weight
- **Medium** (`GoogleSans-Medium.ttf`) - Medium weight
- **SemiBold** (`GoogleSans-SemiBold.ttf`) - Semi-bold weight
- **Bold** (`GoogleSans-Bold.ttf`) - Bold weight

## Usage

The fonts are loaded in `app/_layout.tsx` and can be used in components like this:

```typescript
fontFamily: 'GoogleSans-Regular'
fontFamily: 'GoogleSans-Medium'
fontFamily: 'GoogleSans-SemiBold'
fontFamily: 'GoogleSans-Bold'
```
