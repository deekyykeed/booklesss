# RobotoCondensed Font Family

This folder contains the RobotoCondensed font family used for section titles in the app.

## Available Weights

- **Regular** (`RobotoCondensed-Regular.ttf`) - Default weight
- **Medium** (`RobotoCondensed-Medium.ttf`) - Medium weight
- **SemiBold** (`RobotoCondensed-SemiBold.ttf`) - Semi-bold weight
- **Bold** (`RobotoCondensed-Bold.ttf`) - Bold weight

## Usage

The fonts are loaded in `app/_layout.tsx` and can be used in components like this:

```typescript
fontFamily: 'RobotoCondensed-Regular'
fontFamily: 'RobotoCondensed-Medium'
fontFamily: 'RobotoCondensed-SemiBold'
fontFamily: 'RobotoCondensed-Bold'
```
