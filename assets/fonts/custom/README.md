# Custom Fonts

Place your custom font files (.ttf or .otf) in this folder.

## Supported Font Formats

- `.ttf` (TrueType Font)
- `.otf` (OpenType Font)

## Naming Convention

Use descriptive names for your font files:

```
assets/fonts/custom/
├── Montserrat-Regular.ttf
├── Montserrat-Bold.ttf
├── Montserrat-SemiBold.ttf
├── Roboto-Regular.ttf
├── Roboto-Italic.ttf
└── YourCustomFont-Regular.ttf
```

## After Adding Fonts

1. Add your fonts to this folder
2. Follow the instructions in `HOW_TO_USE_CUSTOM_FONTS.md` at the project root
3. Restart your development server with cache cleared: `npm start -- --clear`
